```
# 📌 Project Overview
```

```
This project simulates a real-world Employee Management Database where different
SQL operations and database testing techniques are performed.
```

```
The objective of this project is to demonstrate:
```

```
- Database Design
- SQL Query Writing
- Database Validation
- Constraint Testing

- Bug Simulation
```
