Project Name
-------------
Manual Database Testing Project using MySQL

Description
------------
This project demonstrates manual database testing using MySQL Workbench.
It covers database design, SQL query writing, data validation, business rule
testing, constraint testing, bug simulation, test case preparation, and bug
reporting.

The project follows a real-world QA workflow where data is verified against
database constraints and business requirements.
