1. PROJECT OVERVIEW
===============================================================================

The Manual Database Testing Project demonstrates database creation, SQL query
execution, database validation, constraint testing, business rule testing,
bug simulation, test case preparation, and bug reporting.

This project simulates an Employee Management System where employees belong
to different departments.
